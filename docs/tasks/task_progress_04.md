## 使用指南

### 启动完整开发环境
1. VSCode → F5 或运行和调试
2. 选择 "完整开发环境 (后端+Web)"
3. 自动启动:
   - 后端API: http://localhost:8000
   - Web界面: http://localhost:8080
   - API文档: http://localhost:8000/docs

### Node.js 开发
```bash
cd web
npm install          # 安装依赖
npx vite --host     # 启动开发服务器
```

### 健康监控
```bash
# 独立运行健康检查
PYTHONPATH=/workspaces/my_ai_popup_project python3 scripts/start_health_monitor.py --verbose
```
