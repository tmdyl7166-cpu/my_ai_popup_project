# AISP 实时健康监控启动配置

## 依赖安装状态
✅ 所有核心依赖已正确安装:
- fastapi 0.128.0
- uvicorn 0.40.0  
- Flask 3.1.2
- psutil 7.2.1
- numpy 2.2.6
- opencv-python 4.12.0.88

## 服务端口
| 服务 | 端口 | 说明 |
|------|------|------|
| 后端API | 8000 | FastAPI主服务 |
| 健康Web | 8080 | 健康检查界面 |

## 启动命令

### 方式一：使用启动脚本
```bash
cd /workspaces/AISP
python3 scripts/start_health_monitor.py
```

### 方式二：分别启动
```bash
# 终端1：启动后端API
cd /workspaces/AISP
python3 -m backend.unified_api

# 终端2：启动健康检查Web
cd /workspaces/AISP  
python3 scripts/health_check.py --web --port 8080
```

## 访问地址
- API文档: http://localhost:8000/docs
- 健康检查: http://localhost:8000/health
- 实时监控: http://localhost:8080/
- 状态API: http://localhost:8080/api/status
