## 🔧 部署环境要求

### 硬件要求
- **CPU**: Intel i5-8400 / AMD Ryzen 5 2600 或更高
- **内存**: 16GB RAM (推荐32GB)
- **GPU**: NVIDIA GTX 1060 6GB / AMD RX 580 8GB 或更高
- **存储**: 500GB SSD 可用空间

### 软件环境
- **操作系统**: Windows 10/11, Ubuntu 20.04+
- **Python**: 3.8-3.11
- **CUDA**: 11.8+ (可选, 用于GPU加速)
- **依赖包**: 详见 `requirements.txt`
